# Velocentrax_MV_099

This project builds a fully integrated omni-channel analytics framework to measure how promotional activities influence healthcare provider (HCP) prescribing behavior and territory performance.

The system combines multiple promotional channels into a unified analytical layer and generates insights for:

  1. Commercial strategy teams

  2. National business leadership

  3. Field force optimization

  4. Marketing performance measurement
